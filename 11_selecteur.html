aide memoire des selecteurs CSS déja vus

  • * selectionne tous les elements de la page HTML
  • .ma-jolie-navselectionne tous les elements ayant le nom ayant le nom de class "ma-jolie-nav
  • #mon-unique-idselectionne tout les éléments ayant le nom de mon id "mon-unique-id"
  • nav aselectionne les liens a inclus/imbriqué dans la balise nav
  • nav a: hover selectionne tous les tous les liens de la nav et active une animation lors du survol de la souris

proprieté CSS utilisé dans cette page

decrivez leur usages et leurs valeurs possible

  1. background: couleur du fond de l'element
  2. color: couleur de la police
  3. float: effet flottant
  4. font-family: police
  5. font-size: taille de la police
  6. height: hauteur de l'element
  7. left: position a partir de la gauche de la fenetre
  8. line height: hauteur de l'interligne
  9. margin: marge exterieur
  10. padding: marge interieur
  11. position: position de l'element dans la page (on a utilisé que la valeur (fixed) pour l'instant
  12. text-align: alignement du texte
  13. text-decoration: changement des valeurs herités (ici le souligné qui est standard pour les liens sous tous les navigateurs est supprimé)
  14. top: position a partir du haut de la fenetre
  15. transition: temps de transition lors d'une animation (ici sur les liens de la barre de menu)
  16. width: largeur de l'element par rapport a la taille de la fenetre si on indique une valeur relative (%, em)
